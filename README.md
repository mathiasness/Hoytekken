## Spillbeskrivelse

Klassisk 2D-fighting spill i tekken og Street Fighter-stil.

## Om koden

- Kompileres med `mvn package`.
- Tester kjøres med `mvn test`
- Kjøres med `java -jar target/hoytekken-app-1.0-SNAPSHOT-fat.jar`
- Krever Java 17 eller senere
